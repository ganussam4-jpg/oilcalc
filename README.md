# OILCALC

Полевой калькулятор технолога УЭЦН для Android. Все формулы — из регламента
по добыче нефти, без сторонних коэффициентов и интернета.

## Возможности

- 8 разделов: защиты СУ, приток/охлаждение, уровни/напор, кабель/мощность, ПРГ/АПВ, газ, геометрия, конвертер единиц.
- 27 расчётов + 7 конвертеров (давление, длина, объём, дебит, температура, плотность, вязкость, мощность).
- Тёмная тема для работы на ярком солнце, крупные тач-кнопки, масштабируемый шрифт.
- Полностью офлайн: интернет не требуется ни при установке, ни в работе.

## Сборка APK

APK собирается автоматически через GitHub Actions при каждом push в `main`.
См. вкладку **Actions** в репозитории → последний успешный запуск → раздел
**Artifacts** → скачать `oilcalc-apk`.

Первая сборка ~30-45 мин, повторные ~5-10 мин (благодаря кэшу).

## Локальный запуск (для тестирования на ПК)

```bash
pip install kivy==2.3.0
python main.py
```

## Структура

- `main.py` — GUI на Kivy
- `calc_engine.py` — чистые формулы расчётов
- `buildozer.spec` — конфигурация сборки APK
- `.github/workflows/build-apk.yml` — автосборка в облаке
